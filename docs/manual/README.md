# Tortoise-UFH — user manual

The full end-user manual (installation, configuration, the sidebar panel, tuning and the flag
dictionary) is maintained per language in this folder:

- **Polski** — [`pl.md`](pl.md)
- **Deutsch** — [`de.md`](de.md)

English readers: the [project README](../../README.md) is the English overview. A standalone
English manual is not split out here yet.

Section numbering is kept consistent across every translation, so a reference such as "§12"
points to the same section in each language file.
